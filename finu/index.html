How to Use the Prototypes

Main Prototype Features

  • Click hamburger menu (mobile) to open/close sidebar
  • Click financial tools in sidebar to open feature pages
  • Use tool selector bar to switch between features
  • Type in message input and click send button
  • Click quick suggestions to populate input field

Testing Responsive Design

  • Desktop (1024px+): Full sidebar always visible
  • Tablet (768-1023px): Collapsible sidebar
  • Mobile (<768px): Sidebar as overlay
  • Resize browser window to test responsive behavior
